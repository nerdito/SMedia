# Viewing Media

The Media Viewer app provides a simple interface for browsing and playing video files from your selected folder.

## Video Library Screen

The main screen displays all video files from your selected folder in a grid or list view. Each video shows:

- **Thumbnail**: Auto-generated preview of the video
- **Title**: Name of the video file
- **Duration**: Length of the video
- **Date**: When the video was last modified

## Playing Videos

1. **Tap** on any video thumbnail to start playback
2. The video opens in full-screen mode
3. Use on-screen controls to:
   - Play/Pause
   - Seek forward/backward
   - Adjust volume
   - Toggle fullscreen
4. **Swipe down** or tap back to return to the library

## Supported Formats

The app supports all common video formats including:
- MP4
- MKV
- AVI
- MOV
- WebM
- 3GP

## Sorting and Filtering

Videos are automatically sorted by date modified. The app uses Android's MediaStore to scan and display videos from your selected folder.
