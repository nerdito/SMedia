# Configuration

The Settings screen allows you to configure app behavior and security settings. Access requires entering your 4-digit PIN.

## Accessing Settings

1. Tap the **gear icon** in the top-right corner of the main screen
2. Enter your 4-digit PIN
3. Configure desired options

## PIN Security

### Setting Up PIN

On first launch, you'll be prompted to create a 4-digit PIN. This PIN is required to access:
- Settings screen
- Changing the media folder

### Changing PIN

1. Go to Settings
2. Tap "Change PIN"
3. Enter current PIN
4. Enter new PIN and confirm

### Resetting PIN

If you forget your PIN, you'll need to clear app data:
- Go to Android Settings > Apps > Media Viewer
- Tap "Clear Data" or "Clear Storage"

## Folder Selection

### Selecting a Media Folder

1. In Settings, tap "Select Folder"
2. Use Android's Storage Access Framework to browse
3. Choose the folder containing your videos
4. Tap "Allow" to grant access

### Changing the Folder

Repeat the folder selection process to change the media source folder.

## App Information

The Settings screen also displays:
- App version
- Storage permissions status
